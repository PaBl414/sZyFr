# Caesar Cipher Encryption Project

## Overview
The Caesar Cipher is a simple yet effective encryption technique used for encoding messages. This project implements the Caesar Cipher algorithm, which shifts the letters of the alphabet by a fixed number.

## Features
- Encrypt and decrypt messages using the Caesar Cipher algorithm
- User-friendly interface for inputting data
- Supports both uppercase and lowercase letters
- Customizable shift value for encryption and decryption

## How to Use
1. Enter the message you wish to encrypt or decrypt.
2. Specify the shift value (e.g., 3 for a right shift).
3. Click on the encrypt or decrypt button.
4. Review your results in the provided output field.

## Example
- Input Message: FRANEK-TO-CFEL
- Shift Value: 3
- Encrypted: KHOOR
- Decrypted: HELLO

## Installation
To run this project locally:
1. Clone the repository: `git clone https://github.com/PaBl414/sZyFr.git`
2. Navigate to the project directory: `cd sZyFr`
3. Run the application using your preferred method (e.g., using a web server if applicable).
